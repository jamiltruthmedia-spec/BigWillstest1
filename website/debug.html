Common Issues:

  • CSS not loading: Check file path: assets/css/style.css
  • Images missing: Add images to assets/images/ folder
  • Browser cache: Hard refresh (Ctrl+F5 or Cmd+Shift+R)
  • File permissions: Make sure CSS file is readable

File Structure Check:

Current path: website/assets/css/style.css

HTML file: website/index.html

CSS link in HTML: <link rel="stylesheet" href="assets/css/style.css">

Quick Fix

1. Open browser developer tools (F12)

2. Check Console tab for errors

3. Check Network tab to see if style.css loads

4. If CSS loads but looks unstyled, check for missing images
